===Math Stuff Program===
Written by: Lars Fyhr (lcf597)

Instructions:

1. Unzip the contents of this file into project directory (do not need the readme.txt)
    -includes makefile and mathstuff.c

2. Type "make" into the command prompt while in the project directory
    -if you "ls" in the directory, you should now have a "math_stuff" executable, as well as an object file

3. Enter the command "./math_stuff" into the command prompt while still in project directory to launch the program
